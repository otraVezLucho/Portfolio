¿Qué es el DOM?

El DOM (Document Object Model, o Modelo de Objetos del Documento) es una representación estructurada de un documento HTML (o XML) en forma de un árbol de objetos. Es una interfaz que permite a los programas (como JavaScript) interactuar con los elementos de una página web de manera dinámica. En otras palabras, el DOM es como un puente entre tu código JavaScript y los elementos HTML de tu página, permitiéndote leer, modificar, agregar o eliminar elementos, estilos, atributos, y más.

Características clave del DOM:

  • Estructura de árbol: Cada elemento HTML (como <div>, <p>, <button>) se convierte en un nodo en un árbol jerárquico. Los nodos pueden ser elementos, texto, atributos, etc.
  • Dinámico: Puedes cambiar el contenido, la estructura y los estilos de la página en tiempo real.
  • Interfaz de programación: El DOM proporciona métodos y propiedades para manipular la página, como cambiar el texto de un párrafo o agregar un nuevo botón.

Ejemplo visual del árbol DOM:

Para un HTML simple como este:


<!DOCTYPE html>
<html>
  <head>
    <title>Mi Página</title>
  </head>
  <body>
    <h1>Bienvenido</h1>
    <p id="texto">Hola, mundo!</p>
  </body>
</html>

El DOM genera un árbol como este:


Document
  └── html
      ├── head
      │   └── title
      │       └── Texto: "Mi Página"
      └── body
          ├── h1
          │   └── Texto: "Bienvenido"
          └── p (id="texto")
              └── Texto: "Hola, mundo!"

Cada elemento (html, head, body, h1, p) es un nodo elemento, y el texto dentro de ellos es un nodo de texto.

¿Cómo funciona el DOM?

Cuando un navegador carga una página web:

  1. pronosticar el HTML: El navegador lee el código HTML y lo convierte en una estructura de árbol en memoria (el DOM).
  2. Acceso desde JavaScript: JavaScript puede acceder a este árbol a través del objeto global document, que representa el DOM completo.
  3. Manipulación: Usando métodos y propiedades del DOM, puedes modificar elementos, sus atributos, estilos, contenido o estructura.

El DOM actúa como un intermediario: JavaScript le dice qué hacer, y el navegador actualiza la página para reflejar los cambios.

Guía para interactuar con el DOM

Aquí tienes una guía práctica con los conceptos y métodos más comunes para trabajar con el DOM, junto con ejemplos que puedes probar.

1. Acceder a elementos del DOM

Para manipular elementos, primero necesitas seleccionarlos. Aquí están los métodos más usados:

  • document.getElementById(id): Selecciona un elemento por su id.
  • document.querySelector(selector): Selecciona el primer elemento que coincida con un selector CSS (como .clase, #id, div).
  • document.querySelectorAll(selector): Devuelve una lista de todos los elementos que coincidan con el selector.
  • document.getElementsByClassName(class): Selecciona elementos por su clase.
  • document.getElementsByTagName(tag): Selecciona elementos por su etiqueta (como p, div).

Ejemplo:


<p id="texto">Hola, mundo!</p>
<button class="boton">Clic aquí</button>
  

// Seleccionar por ID
const parrafo = document.getElementById("texto");
console.log(parrafo); // <p id="texto">Hola, mundo!</p>

// Seleccionar por clase
const boton = document.querySelector(".boton");
console.log(boton); // <button class="boton">Clic aquí</button>

// Seleccionar todos los párrafos
const parrafos = document.querySelectorAll("p");
console.log(parrafos); // NodeList con todos los <p>

2. Modificar contenido de elementos

Puedes cambiar el contenido de un elemento usando propiedades como:

  • element.textContent: Cambia el texto puro de un elemento.
  • element.innerHTML: Cambia el contenido HTML (puedes agregar etiquetas).
  • element.value: Cambia el valor de elementos de formulario (como <input>).

Ejemplo:


<p id="texto">Hola, mundo!</p>
  

const parrafo = document.getElementById("texto");
parrafo.textContent = "¡Texto cambiado!"; // Cambia el texto
parrafo.innerHTML = "<strong>Texto en negrita</strong>"; // Cambia a HTML

3. Modificar estilos

Puedes cambiar los estilos CSS de un elemento accediendo a la propiedad style.

Ejemplo:


<p id="texto">Hola, mundo!</p>
  

const parrafo = document.getElementById("texto");
parrafo.style.color = "blue";
parrafo.style.fontSize = "20px";
parrafo.style.backgroundColor = "lightgray";

Nota: Los nombres de propiedades CSS con guiones (como font-size) se convierten a camelCase (fontSize) en JavaScript.

4. Modificar atributos

Puedes leer o cambiar atributos HTML con:

  • element.getAttribute(nombre): Obtiene el valor de un atributo.
  • element.setAttribute(nombre, valor): Establece un atributo.
  • element.removeAttribute(nombre): Elimina un atributo.

Ejemplo:


<img id="imagen" src="foto.jpg" alt="Una foto">
  

const imagen = document.getElementById("imagen");
console.log(imagen.getAttribute("src")); // "foto.jpg"
imagen.setAttribute("src", "nueva-foto.jpg"); // Cambia la imagen
imagen.removeAttribute("alt"); // Elimina el atributo alt

5. Crear y agregar elementos

Puedes crear nuevos elementos y añadirlos al DOM:

  • document.createElement(tag): Crea un nuevo elemento.
  • element.appendChild(nodo): Agrega un nodo como hijo.
  • element.removeChild(nodo): Elimina un nodo hijo.
  • element.append(nodo): Agrega nodos o texto al final.

Ejemplo:


<div id="contenedor"></div>
  

const contenedor = document.getElementById("contenedor");
const nuevoParrafo = document.createElement("p");
nuevoParrafo.textContent = "Soy un nuevo párrafo";
contenedor.appendChild(nuevoParrafo); // Agrega el párrafo al div

6. Eliminar elementos

Puedes eliminar elementos con remove() o removeChild().

Ejemplo:


<p id="texto">Hola, mundo!</p>
  

const parrafo = document.getElementById("texto");
parrafo.remove(); // Elimina el párrafo del DOM

7. Manejar eventos

Los eventos (como clics, cambios en formularios, etc.) son clave para hacer que tu página sea interactiva. Usa addEventListener para asociar funciones a eventos.

Ejemplo:


<button id="boton">Clic aquí</button>
<p id="texto">Hola</p>
  

const boton = document.getElementById("boton");
const parrafo = document.getElementById("texto");

boton.addEventListener("click", () => {
  parrafo.textContent = "¡Clic detectado!";
  parrafo.style.color = "red";
});

Tipos comunes de eventos:

  • click: Clic en un elemento.
  • change: Cambio en un input.
  • mouseover: El mouse pasa sobre un elemento.
  • keydown: Se presiona una tecla.

8. Navegar por el árbol DOM

Puedes moverte entre nodos usando propiedades como:

  • element.parentElement: El elemento padre.
  • element.children: Lista de elementos hijos.
  • element.nextElementSibling: El siguiente elemento hermano.
  • element.previousElementSibling: El elemento hermano anterior.

Ejemplo:


<div id="contenedor">
  <p>Primero</p>
  <p>Segundo</p>
</div>
  

const contenedor = document.getElementById("contenedor");
console.log(contenedor.children); // Lista con los dos <p>
console.log(contenedor.children[0].nextElementSibling); // Segundo <p>

Ejemplo práctico completo

Aquí tienes un ejemplo que combina varias técnicas del DOM para crear una lista dinámica:


<!DOCTYPE html>
<html>
<head>
  <title>Lista Dinámica</title>
  <style>
    .item { padding: 10px; background-color: lightblue; margin: 5px; }
    .rojo { background-color: lightcoral; }
  </style>
</head>
<body>
  <h1>Mi Lista</h1>
  <input type="text" id="entrada" placeholder="Escribe un elemento">
  <button id="agregar">Agregar</button>
  <button id="limpiar">Limpiar</button>
  <ul id="lista"></ul>

  <script>
    const entrada = document.getElementById("entrada");
    const botonAgregar = document.getElementById("agregar");
    const botonLimpiar = document.getElementById("limpiar");
    const lista = document.getElementById("lista");

    // Agregar elemento a la lista
    botonAgregar.addEventListener("click", () => {
      const texto = entrada.value.trim();
      if (texto) {
        const nuevoItem = document.createElement("li");
        nuevoItem.textContent = texto;
        nuevoItem.className = "item";
        // Cambiar color al hacer clic
        nuevoItem.addEventListener("click", () => {
          nuevoItem.classList.toggle("rojo");
        });
        lista.appendChild(nuevoItem);
        entrada.value = ""; // Limpiar input
      }
    });

    // Limpiar la lista
    botonLimpiar.addEventListener("click", () => {
      lista.innerHTML = ""; // Elimina todos los hijos
    });
  </script>
</body>
</html>

Qué hace este ejemplo:

  • Permite agregar elementos a una lista <ul> desde un <input>.
  • Cada elemento agregado tiene una clase CSS (item) y cambia de color al hacer clic.
  • El botón "Limpiar" elimina todos los elementos de la lista.
  • Usa métodos como createElement, appendChild, classList.toggle, y eventos como click.

Consejos para volverte experto en el DOM

  1. Practica con proyectos pequeños:
    • Crea una calculadora que actualice resultados en la página.
    • Haz un formulario que valide datos en tiempo real.
    • Construye una lista de tareas (to-do list) con funciones para agregar, eliminar y marcar tareas.
  2. Entiende los selectores CSS:
    • Aprende a usar querySelector y querySelectorAll con selectores como .clase, #id, [data-atributo], etc. Esto te dará flexibilidad para seleccionar elementos.
  3. Domina los eventos:
    • Experimenta con eventos como input, submit, mouseover, keydown. Usa addEventListener en lugar de atributos como onclick para mantener el código limpio.
  4. Usa herramientas de desarrollo:
    • Abre las DevTools del navegador (F12) y usa la pestaña "Elements" para inspeccionar el DOM. También usa la consola para probar comandos como document.querySelector.
  5. Evita errores comunes:
    • No uses innerHTML para todo, ya que puede ser inseguro (vulnerable a inyecciones de código) y lento. Prefiere textContent para texto puro.
    • Asegúrate de que los elementos existan antes de manipularlos (p.ej., verifica si document.getElementById devuelve null).
  6. Explora bibliotecas modernas:
    • Una vez domines el DOM puro, prueba frameworks como React o Vue.js, que abstraen el manejo del DOM pero se basan en los mismos conceptos.
  7. Recursos para aprender más:
    • MDN Web Docs: La referencia definitiva para el DOM (busca "DOM" o "JavaScript" en developer.mozilla.org).
    • FreeCodeCamp: Tutoriales prácticos de JavaScript y DOM.
    • JavaScript.info: Explicaciones detalladas sobre el DOM y eventos.

Ejercicios para practicar

  1. Cambiar colores: Crea un botón que cambie el color de fondo de la página al hacer clic.
  2. Contador: Haz un contador que incremente o decremente un número mostrado en un <p> con dos botones (+ y -).
  3. Galería de imágenes: Crea una galería donde al hacer clic en una miniatura, cambie la imagen principal.
  4. Validación de formulario: Valida un formulario en tiempo real (p.ej., verifica que un campo de correo tenga un formato válido).

Resumen

El DOM es la representación de tu página web como un árbol de nodos que puedes manipular con JavaScript. Usando métodos como getElementById, querySelector, createElement, y addEventListener, puedes hacer que tu página sea interactiva. Practica con ejemplos pequeños, inspecciona el DOM en las DevTools, y construye proyectos para ganar experiencia. Con el tiempo, dominarás el DOM y estarás más cerca de ser un experto en desarrollo web. ¡Sigue adelante y experimenta con el código!
